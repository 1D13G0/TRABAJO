En informática, se conoce como lenguaje de programación a un programa destinado a la construcción de otros programas informáticos

  1. Python:Es de código abierto, con una sintaxis simple y sencilla de entender.
  2. Java: Este lenguaje orientado a objetos funciona independientemente de la plataforma de hardware, entonces, el código escrito en una máquina corre en otra, sin problemas.
  3. JavaScript: Como es interpretado, no necesita compilación, entonces los navegadores leen el código y ejecutan las acciones indicadas.
  4. PHP:PHP es un lenguaje de programación multipropósito que permite desarrollar aplicaciones del lado del servidor, garantizando una excelente comunicación con la web.
